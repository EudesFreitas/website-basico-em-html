Exemplos de Tags utilisadas no HTML

Índice

As principais tags HTML são aquelas que compõem a estrutura básica de uma página web, permitindo a criação de conteúdo, títulos, parágrafos, listas, imagens e links. Entre elas, destacam-se as tags para estruturar a página (<html>, <head>, <body>), títulos (<h1> a <h6>), parágrafos (<p>), imagens (<img>), links (<a>), listas (<ul>,<ol>,<li>), e algumas tags semânticas como <article>, <section>, <header>, <footer>.

Tags de Estrutura:

<html>: Define a raiz de um documento HTML.
<head>: Contém informações sobre o documento, como título, metadados e links para estilos e scripts.
<body>: Define o conteúdo visível do documento.

Tags para Conteúdo:

  • <h1> a <h6>: Define títulos de diferentes níveis de importância.
  • <p>: Define um parágrafo de texto.
  • <a>: Cria um hiperlink para outra página ou seção do documento.
  • <img>: Inserte uma imagem na página.
  • <ul> e <ol>: Criam listas não ordenadas e ordenadas, respectivamente.
  • <li>: Define um item de lista dentro de uma lista ordenada ou não ordenada.

Tags Semânticas:

<article>: Define um conteúdo independente, como um artigo de blog ou postagem.
<section>: Define uma seção dentro do documento, geralmente com um título.
<header>: Define o cabeçalho de uma página ou seção.
<footer>: Define o rodapé de uma página ou seção.
<nav>: Define a área de navegação da página.

Outras Tags Importantes:

<br>: Insere uma quebra de linha.
<hr>: Cria uma linha horizontal.
<div> e <span>: Usados para agrupar conteúdo para aplicar estilos (div é de bloco e span é inline). É importante notar que o HTML5 introduziu diversas tags semânticas para melhorar a estrutura e o significado do conteúdo. Compreender o uso dessas tags é fundamental para criar páginas web bem estruturadas e acessíveis.
